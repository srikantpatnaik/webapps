# Web Apps Collection

This repository contains a collection of web applications hosted on GitHub Pages.

## Applications

- [FIRE Calculator](fire/fire_calculator.html) - A Financial Independence, Retire Early calculator to help determine when you can achieve financial independence based on your current financial situation and projected expenses.
- [Age Logic Test](fire/test_age_logic.html) - Test page for verifying the age calculation logic in the FIRE calculator
- [Interactive Age Logic Test](fire/test_logic.html) - Interactive test page for the age calculation logic in the FIRE calculator

For more details about the FIRE calculator, see the [FIRE Calculator README](fire/README.md).
