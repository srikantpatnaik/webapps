About Image Resizer

Features

  • Paste images directly from clipboard (Ctrl+V)
  • Drag and drop image files anywhere on the page
  • Enter image URLs in the input field
  • Continuous resize slider (10% to 200%)
  • Real-time crop preview with rectangle selection
  • See exact resolution of both original and resized images

How to Use

  1. Open the application in your browser
  2. Paste an image with Ctrl+V (or Cmd+V on Mac) anywhere on the page
  3. Or drag and drop an image file anywhere on the page
  4. Or enter an image URL in the text field and click "Load Image"
  5. Adjust the slider to set your desired resize factor (10% to 200%)
  6. Select a rectangular area on the image to crop
  7. See the cropped and resized result in real-time
