Video Player Test Page

This page allows you to test the video player functionality by opening it in an iframe below.

Try uploading a video file and using the keyboard shortcuts:

  • Space: Play/Pause
  • →: Seek Forward 10s
  • ←: Seek Backward 10s
  • ↑: Volume Up 10%
  • ↓: Volume Down 10%
  • M: Mute/Unmute
  • F: Toggle Fullscreen
  • Esc: Close Video
